**Голосовой помощник с двумя девушками** — **Чу Чу** (18 лет, косплей-модель) и **Мэй** (22 года, крафтерша). Ассистент распознаёт голос, отвечает через нейросеть Qwen 2.5-7B (локально, без интернета), поддерживает фоновые диалоги между персонажами и автоматическое исправление опечаток.

## ✨ Возможности

| Возможность | Описание |
|-------------|----------|
| 🎤 Голосовое управление | Говорит и слушает через микрофон |

## 🔐 Голосовая идентификация

При первом запуске ассистент запомнит ваш голос и имя:

1. **Первый запуск** — запись образца голоса и имени владельца
2. **Последующие запуски** — автоматическое распознавание владельца по голосу
3. **Гостевой режим** — если голос не распознан, включится бета-режим (SFW)
4. **Кодовая фраза** — гости могут открыть открытый режим (NSFW) фразой "открой мне мир"

## 💻 Управление компьютером

| Команда | Действие |
|---------|----------|
| `выключи компьютер` / `shutdown` | Выключение ПК через 30 сек |
| `отмена выключения` / `не выключай` | Отмена выключения |
| `перезагрузи компьютер` / `reboot` | Перезагрузка ПК через 30 сек |
| `спящий режим` / `усыпи компьютер` | Спящий режим |
| `заблокируй компьютер` / `lock` | Блокировка экрана |
| `увеличь громкость` / `громче` | +10% громкости |
| `уменьши громкость` / `тише` | -10% громкости |
| `выключи звук` / `mute` | Отключение звука |
| `включи звук` / `unmute` | Включение звука |

Поддерживаются: блокнот, калькулятор, проводник, командная строка, диспетчер задач, панель управления, браузеры (Chrome, Opera, Firefox), офисные программы (Excel, Word, PowerPoint), мессенджеры (Telegram, Discord, WhatsApp), медиаплееры (VLC), редакторы (Photoshop, VS Code, PyCharm), игровые платформы (Steam, Epic Games) и многие другие.
